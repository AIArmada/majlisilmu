# MajlisIlmu: Visitor Guide

## What Is MajlisIlmu?
MajlisIlmu is a community platform that helps people discover, verify, and join Islamic learning events.

It brings together information that is usually scattered across posters, chat groups, social media, and word of mouth into one searchable place.

## What Problem Does It Solve?
People frequently face:
- Unclear event details (time, venue, speaker, status).
- Last-minute changes not reaching everyone.
- Difficulty finding events by topic or location.
- Trust concerns around donation links and organizer authenticity.

MajlisIlmu addresses this with structured event listings, moderation workflows, and clearer organizer/event context.

## Who Benefits?

### 1. Attendees and families
- Find events by date, location, topic, speaker, and format.
- Discover nearby events and upcoming sessions quickly.
- Save events, mark interest, and track what matters to you.

### 2. Institutions and organizers
- Publish events in a standard format that is easier to share.
- Keep event details updated from one source of truth.
- Manage event records and supporting details over time.

### 3. Speakers
- Be discoverable by topic and participation history.
- Stay connected to institutions and event pages.
- Build richer public profiles with biography, avatar/main image, and affiliated institution role.

### 4. Moderators and administrators
- Review submissions and maintain quality.
- Handle reports and reduce misinformation.
- Keep community trust high.

## What Makes It Interesting?
- Prayer-time-aware scheduling support.
- Public and admin workflows in one coherent system.
- Rich media support (poster, gallery, profile images, etc.).
- Speaker profiles support rich biography content and featured profile images.
- Search with geolocation and filtering capabilities.
- Built-in moderation lifecycle (pending, approved, needs changes, rejected).
- Saved-search digest notifications for recurring discovery.

## Typical Visitor Journeys

### A. “I want to find something near me tonight”
1. Open the homepage.
2. Use search or near-me discovery.
3. Filter by date, language, event type, or topic.
4. Open event details and navigate to location or join online.

### B. “I want to follow a specific topic or speaker”
1. Search for the topic, or search for the speaker by name.
2. Open the speaker profile to see upcoming and past events.
3. Save the events you like, or revisit the search later for new updates.

### C. “I want to submit an event”
1. Open the submit-event page.
2. Fill event details (time, format, organizer, tags, media), including quick-adding a speaker with biography, affiliated institution role, and main profile image.
3. Submit for moderation.
4. Track publication after review.

## Why Trust It?
MajlisIlmu uses:
- Structured submissions (instead of free-form posts only).
- Moderation review workflows for publishing decisions.
- Reporting channels for community feedback.
- Explicit status signaling (pending, approved, etc.) so users can judge confidence.

## How You Can Help

### If you are a visitor
- Report outdated or suspicious content.
- Share event links with your local community.
- Save searches and use feedback channels.

### If you are an organizer or institution
- Keep event details complete and updated.
- Use clear titles, accurate timings, and location data.
- Add valid media and organizer references.

### If you are a contributor (technical or product)
- Improve data quality and consistency.
- Help optimize search, moderation tools, and notification workflows.
- Contribute tests, bug fixes, and UX improvements.

## Frequently Asked Questions

### Is this only for physical events?
No. The platform supports physical, online, and hybrid events.

### Can I find events by audience suitability?
Yes. Listings support filters such as age group, gender restrictions, and children-allowed settings when provided.

### Can I register for events?
Some events include registration when organizers enable it.

### Can I find events by a specific speaker?
Yes. Search for the speaker by name, then open the speaker profile to view upcoming and past events tied to that speaker.

### Can I trust every listing?
Use the event status and details as guidance. The system supports moderation and reporting, but users should still verify critical details when needed.

## How To Start Right Now
1. Browse upcoming events from the homepage.
2. Use filters to narrow by what matters to you.
3. Open event pages for full details and navigation.
4. If you have an event to share, submit it through the public submission flow.

MajlisIlmu exists to make beneficial knowledge easier to find, easier to trust, and easier to attend.
